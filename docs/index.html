TUIテキストエディタ

CirPathは、Rustで書かれたシンプルかつエレガントなTUIテキストエディタ。極限まで無駄を削ぎ落とした「ボーダーレス」なデザインと、3つのウィンドウによる革新的な操作性を提供します。

なぜ CirPath なのか？

📐

3つのシームレスな窓

Editor、File Browser、Menu の3面構造。Ctrl+E / Ctrl+Q で視覚的・直感的に高速切り替えが可能。

✨

徹底したミニマリズム

nanoにインスパイアされた、罫線（ボーダー）のない極限デザイン。タイトルバーと反転色のステータスバーのみで画面を広く使えます。

🌐

環境非依存 & 多言語対応

絵文字を排除し、JIS規格文字とASCIIのみで構築。崩れる心配はありません。さらに英語・日本語をメニューから即時変更可能。

ビルドと実行

Rust環境があれば、数秒で最強の軽量エディタがあなたのターミナルに。依存関係に縛られないネイティブスピードを体感してください。

Bash 
cargo build --release
./target/release/cirpath

拡張性の高い設計

各スクリーンはモジュールとして完全に独立。新機能の追加も既存モジュールを破壊しません。

editor.rs → エディタ描画とテキスト入力制御
browser.rs → 高速ファイルシステム探索
menu.rs → 設定とキーバインド一覧
