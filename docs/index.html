シンプルな操作感、極限の軽快さ。

CirPath は、nano / pico にインスパイアされた Rust 製 TUI テキストエディタです。 枠線のないフラットなデザインと、瞬時に切り替え可能な日英マルチリンガルインターフェースを特徴としています。

主な特徴

3つの画面、1つのアプリ

エディタ、ファイルブラウザ、設定メニューの3ウィンドウを内蔵。Ctrl+E / Ctrl+Q でシームレスに切り替えて作業が可能です。

環境非依存のクリーンUI

絵文字や特殊記号（外字）を一切排除。標準的なASCIIとJIS文字のみで構成されているため、どの端末エディタでも文字化けせずに美しく表示されます。

日英完全マルチリンガル

ショートカットグリッドから設定、エラーメッセージに至るまで日英両対応。設定画面からリアルタイムで即座に表示言語を変更できます。

安心の視認性設計

端末の背景が黒でも見えづらくならないよう、青色系（Blue/Cyan）を排除。注意やエラーは「赤背景＋白文字」で直感的かつ速やかに伝えます。

ビルド & 実行方法

CirPath は Rust 1.75 以上 のツールチェーンに対応しています。 

# リポジトリをクローンした後にビルドを実行します
cargo build --release

# コンパイルされたバイナリを実行
./target/release/cirpath

■ ツールチェーンに関する注意：

本プロジェクトは、互換性維持のため Cargo.toml 内で依存関係のバージョンを厳密に固定（ピン留め）しています（例: ratatui = "=0.26.3" など）。最新の Rust ツールチェーンをご利用の場合は、必要に応じて cargo update を実行し、これらのバージョンロックを緩和しても安全です。

操作方法（キーバインド）

キーバインド 実行アクション
Ctrl+E 次のウィンドウへ切り替え（エディタ → ブラウザ → メニュー のループ）
Ctrl+Q 前のウィンドウへ切り替え
Ctrl+G ヘルプを表示（設定メニューの「Key Bindings」項目に直接ジャンプ）
キーバインド 実行アクション
任意の文字キー カーソル位置に文字を挿入（日本語IMEはOS標準システムがそのまま機能します）
Enter 改行を挿入
Backspace / Delete カーソルの前/後ろの文字を削除（行の結合も可能）
矢印キー / Home / End / PageUp / PageDown カーソルの移動
Ctrl+S ファイルの保存
Ctrl+O 名前を付けて保存（Save As / Write Out）※プロンプトが表示されます
Ctrl+R 別のファイルの内容を現在位置に挿入（Read File）
Ctrl+W テキストの検索（Where Is）。空欄でEnterを押すと前回の検索をリピート
Ctrl+\ テキスト置換。検索語と置換語を入力し、すべてのマッチを置換
Ctrl+_ 指定した行番号にジャンプ
Ctrl+K 現在の行を切り取り（カット）。連続で押すとクリップボードに累積されます
Ctrl+U カットした行を貼り付け（Paste / UnCut）
Ctrl+C 現在のカーソル位置（行・列）および総行数をステータスバーに表示
Ctrl+X 終了。未保存の変更がある場合は、確認プロンプトが表示されます
Esc 各プロンプト（検索、置換、保存、行ジャンプなど）のキャンセル
キーバインド 実行アクション
Up / Down 選択する項目（ファイル・フォルダ）を移動
Enter / Right 選択したフォルダに入る、またはファイルを開く
Backspace / Left / u 親ディレクトリに移動する
g 絶対パスまたは相対パスを直接入力してジャンプ（現在の位置以外へも移動可能）
n 現在のディレクトリに新規ファイルを作成
N (Shift+n) 現在のディレクトリに新規フォルダを作成
r 選択中のファイルまたはフォルダの名前を変更
d 選択中のファイルまたはフォルダを削除（y/n の確認プロンプトが出ます）
Ctrl+H 隠しファイル（ドットファイル）の表示/非表示を切り替え
Esc 入力プロンプトのキャンセル
キーバインド 実行アクション
Left / Right 左側のカテゴリパネルと、右側の詳細設定パネルの間でフォーカスを切り替え
Up / Down フォーカスしているパネル内で項目を移動
Enter 左パネル：右パネルに移動
右パネル：選択した設定を適用、または有効/無効の切り替え

■ 設定カテゴリについて：

General (言語の変更)、Editor (エディタ動作)、File Browser (ブラウザ動作)、Key Bindings (確認用一覧リスト)、Config File (config.tomlをエディタで直接開いて編集)、About の各カテゴリがあります。変更は即座に反映され、~/.config/cirpath/config.tomlに自動保存されます。
