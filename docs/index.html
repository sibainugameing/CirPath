CirPath

nano / pico にインスパイアされた、Rust製の極限までミニマルなターミナル専用テキストエディタ。

ドキュメントを見る

特徴的な3つのウィンドウ

CirPathは、余計な枠線を排除し、画面を最大限に活用できるボーダーレスデザインを採用しています。以下の3つのウィンドウを Ctrl+E / Ctrl+Q で瞬時に切り替えることができます。

01. Editor

クラシックなnanoスタイルを踏襲。画面下部には常に2行×6列のショートカットキーガイドが表示され、コマンドに迷うことはありません。

02. File Browser

ファイルやフォルダの作成、削除、リネーム、ディレクトリ間の移動をスムーズに行えます。ファイルを選択すると、自動でエディタへと移行します。

03. Menu

Word風の2ペイン構成で直感的に設定を変更可能。変更内容はリアルタイムで反映され、設定ファイルへ自動保存されます。

ビルド & 実行手順

CirPathはRustで書かれており、以下のコマンドで簡単にビルド・実行できます。 

# リリースビルドの作成
cargo build --release

# 実行
./target/release/cirpath

⚠️ ツールチェーンに関する注意

本プロジェクトは、最小のRustツールチェーンとして rustc 1.75 / cargo 1.75 をターゲットにしています。依存関係は Cargo.toml にて厳密に固定されていますが、より新しいRustバージョンをご利用の場合は cargo update で制限を緩和することが可能です。
