Borderless Philosophy

Designed strictly with zero platform-dependent emojis or special fonts. The UI relies only on clean, standard ASCII layout elements and classic reverse-video status lines.

Self-Contained Screens

Each module—Editor, Browser, and Menu—is completely isolated. Easily expand code architecture without fearing broken side-effects across the app.

Immediate State Saves

All config adjustments made in the Word-style two-pane menu write and apply globally instantly, storing to local directories automatically.

Building & Installation

Verify your environment is set up with cargo, copy the build script block below, and launch it directly in your terminal: 

# Build release binaries directly
cargo build --release

# Launch the binary executable
./target/release/cirpath
