CirPath

A minimalist Rust TUI (terminal user interface) text editor inspired by nano and pico.

View Controls

Three Windows, Single App

CirPath avoids cluttered UI elements and boxed borders. Like classic nano, the interface is built from a plain title bar and reverse-video status bars. You can switch seamlessly between three distinct screens using Ctrl+E / Ctrl+Q.

01. Editor

Designed with a classic nano feature set. A 2-row by 6-column shortcut grid is always visible at the bottom of the screen, keeping commands at your fingertips.

02. File Browser

Easily navigate directories and create, delete, or rename files and folders. Opening a file can automatically switch focus back to the Editor.

03. Menu

A Word-style, two-pane settings screen. Instantly toggle preferences and customize your editor. All changes persist automatically.

Building & Running

Compile and run the app directly from your terminal with Cargo: 

# Build the release binary
cargo build --release

# Run the app
./target/release/cirpath

⚠️ Toolchain Note

This project targets a minimum Rust toolchain of rustc 1.75 / cargo 1.75 and pins exact dependency versions in Cargo.toml to ensure compatibility. If you are using a newer Rust toolchain, these pins are safe to relax using cargo update.
