仕様と制限事項

完全な日本語・英語サポート

CirPathは日本語（JISコード範囲内）と英語の完全な2か国語表示に対応しています。メニュー、タイトル、エラー、さらにはPICO風のショートカットガイドに至るまで実行時に一瞬で切り替えが可能です。

※切り替えは「Menu -> General -> Switch display language」から行えます。

アクセシブルなカラーパレット

一部の黒背景ターミナルにおいて、視認性が著しく低下する「ブルー / シアン」系統の色を徹底的に排除しています。配色はシンプルかつ明瞭な色構成に制限されています：

  • 白背景 ＋ 黒文字（通常ステータス、プロンプト）
  • 赤背景 ＋ 白文字（エラー、警告表示）
  • 黄色太字（ファイルブラウザ内のディレクトリ名）
  • ダークグレー（行番号、非フォーカス枠）

エラー表示の挙動

ファイル保存の失敗や検索不一致などのエラーが発生した場合、ステータスバーが自動で「赤背景」に変化します。

[ERROR] File could not be saved. / ファイルを保存できません。

このエラー表示は、何か次のキーが入力された瞬間に自動でクリアされる設計になっており、古いエラーメッセージが画面に残り続ける煩わしさから解放されます。

既知の制限 (未実装の機能)

現時点で、以下の一般的なテキストエディタ機能は実装されていません。これらは既存の editor.rs アーキテクチャの上に、必要に応じて拡張可能です。

  • マルチバッファ / タブ表示
  • シンタックスハイライト
  • アンドゥ (元に戻す) / リドゥ (やり直す)
  • コピー用のテキスト範囲選択 (行全体の切り取り Ctrl+K / 貼り付け Ctrl+U のみ対応)
