キーバインド リファレンス

グローバルキー (全画面共通)

キー入力 アクション
Ctrl+E 次のウィンドウへ切り替え (Editor → Browser → Menu)
Ctrl+Q 前のウィンドウへ切り替え
Ctrl+G ヘルプ - メニュー画面の「Key Bindings」へ直接ジャンプします

Editor 画面 (nanoスタイル)

キー入力 アクション
文字入力 カーソル位置に文字を挿入 (日本語などのIMEはOSが処理します)
Enter 改行を挿入
Backspace / Delete カーソル前/カーソル位置の文字を削除 (行をまたぐ結合に対応)
矢印キー / Home / End / Page Up / Page Down カーソルを移動
Ctrl+S 上書き保存
Ctrl+O 名前を付けて保存 (ファイル名を入力プロンプトで指定)
Ctrl+R 他のファイルの内容を現在のカーソル位置に挿入 (ファイル読み込み)
Ctrl+W 検索。空の状態でEnterを押すと前回の検索をリピート
Ctrl+\ 置換 (検索文字列と置換後文字列を入力し、全一致を置換)
Ctrl+_ 指定した行番号へジャンプ
Ctrl+K 現在の行を切り取り (連続で押すとクリップボードに蓄積されます)
Ctrl+U 最後に切り取ったテキストをカーソル位置に貼り付け
Ctrl+C 現在のカーソル位置 (行/列) および総行数を表示
Ctrl+X 終了 (未保存の変更がある場合は、再度押して確認)
Esc 現在のプロンプト (検索・置換・保存・移動など) をキャンセル

File Browser 画面

キー入力 アクション
Up / Down 選択項目を移動
Enter / Right 選択したディレクトリに入る、または選択したファイルを開く
Backspace / Left / u 親ディレクトリに移動
g 絶対パスまたは相対パスを入力して、その場所へ直接ジャンプ
n カレントディレクトリに新しいファイルを作成
N (Shift+n) カレントディレクトリに新しいフォルダを作成
r 選択中のファイルまたはフォルダの名前を変更
d 選択中のファイル・フォルダを削除 (実行前に y/n の確認あり)
Ctrl+H 隠しファイル (ドットファイル) の表示/非表示を切り替え
Esc 現在のプロンプトをキャンセル

Menu 画面

キー入力 アクション
Left / Right 左ペイン (カテゴリ) と右ペイン (詳細設定) のフォーカスを切り替え
Up / Down フォーカス中のペイン内で選択項目を移動
Enter 左ペインからは右ペインへ移動。右ペインからは設定を適用/切り替え
