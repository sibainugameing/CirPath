Key Bindings Guide

Quickly find keys or shortcuts. Type what you want to do (e.g. "Save", "Ctrl+S") below to search across all views instantly.

Global Controls

Key Action
Ctrl+E Switch to next window (Editor -> Browser -> Menu)
Ctrl+Q Switch to previous window
Ctrl+G Help menu - Jumps directly to "Key Bindings" in Configuration

Editor Screen

Key Action
Printable chars Insert at current cursor position (native OS IME handles Japanese composition)
Enter Insert line break / newline
Backspace / Delete Delete character before/at cursor (merges lines cleanly)
Arrow Keys / Home / End Move cursor positions
Ctrl+S Save current open document
Ctrl+O Save As (Write Out) - prompts target path name
Ctrl+R Insert contents of another document at cursor position (Read File)
Ctrl+W Search matches (Where Is). Retries last search query on empty entry
Ctrl+\ Global search and replace operations
Ctrl+_ Jump straight to line coordinates
Ctrl+K Cut line to clipboard buffer (consecutive commands append items)
Ctrl+U Paste (UnCut) elements back from active buffer
Ctrl+C Display metrics (cursor line/col, total items)
Ctrl+X Exit / Quit (demands confirmation prompts on unsaved changes)
Esc Cancel active input prompts (search/replace/jump)

File Browser

Key Action
Up / Down Scroll selection upwards or downwards
Enter / Right Explore folders or load file focus straight to editor
Backspace / Left / u Ascend back up one directory level
g Direct absolute or relative path navigation jumps
n Spawn new text document in current directory location
N (Shift+n) Create new directory folder
r Rename selected file/folder paths
d Delete directory targets (requires strict y/n validations)
Ctrl+H Toggle visibility of hidden dotfiles
Esc Cancel path inputs and creation prompts

Menu Screen

Key Action
Left / Right Toggle focus between navigation categories and settings list panel
Up / Down Navigate active selections in selected panel lists
Enter Confirm selection or toggle configuration checkboxes
